This game is inspired by "HJKL" by Bi Ran, a Vimscript maze game for beginners to familiarize themselves with Vim's direction keys.

The game further allows switching between Vim and Emacs key bindings and setting of Maze size. To play, a keyboard is needed (obviously).

Written in Elm by Xu Yecheng, Layout and colors are strongly influence by Flatris by Andrey Kuzmin.
