This game is inspired by "HJKL" by Bi Ran, a Vimscript maze game for beginners to familiarize themselves with Vim's direction keys.

The game further allows switching between Vim and Emacs key bindings and setting of Maze size. To play, a keyboard is needed (obviously). As a multiplayer game, you can compete with others as well!

Written in Elm by Xu Yecheng, Layout and colors are strongly influenced by Flatris by Andrey Kuzmin.
